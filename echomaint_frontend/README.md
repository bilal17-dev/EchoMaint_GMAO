# EchoMaint — Frontend

## Stack technique
- React + Vite
- Axios (appels API)
- React Router DOM (navigation)

## Installation

```bash
npm install
```

## Lancement

```bash
npm run dev
```
Serveur disponible sur `http://localhost:5173`
